## Basic Cybersecurity Concepts

### Networking Basics
- OSI Model consists of 7 layers: Physical, Data Link, Network, Transport, Session, Presentation, and Application.
- TCP/IP is the standard communication protocol used on the internet.
- DNS converts domain names into IP addresses.
- HTTP and HTTPS are protocols used for web communication, with HTTPS providing encryption.
- NAT allows multiple devices to share a single public IP address.

### Cryptography Basics
- Symmetric Encryption uses the same key for encryption and decryption.
- Asymmetric Encryption uses a public and private key pair.
- MD5 and SHA256 are hashing algorithms used for data integrity verification.
- SSL/TLS certificates secure communications over HTTPS.

### Security Tools Overview
- Wireshark is used for packet capture and network analysis.
- Nmap is used for network discovery and port scanning.
- Burp Suite is used for web application security testing.
- Netcat is used for network debugging and communication testing.
